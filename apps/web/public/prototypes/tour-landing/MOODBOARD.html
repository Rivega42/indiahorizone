04 — Tone of voice3 примера повторяем · 3 отвергаем

Все примеры русскоязычные — адаптация англоязычного тона под нашу ЦА. Headline + 1–2 строки supporting copy для tour-page hero.

✓ Берём этот тон
«Двенадцать дней в Керале — без спешки, без туристических автобусов, без обязательных селфи у фонтана.»
Black Tomato style — отрицанием, через «не как у всех»
«Мы знаем Кералу как свою — и подберём именно тот ритм, что нужен вам.»
Audley specialist style — concierge-обещание, человеческое
«Чай с фермером в горах Муннара. Лодка по backwaters Аллеппи. Три ночи под звёздами в Текади.»
Original Travel style — конкретные образы вместо прилагательных
✗ Отвергаем
«Откройте магию Индии! Незабываемое путешествие в страну тысячи богов!»
Восклицания, штампы, «магия»/«незабываемое»
«Эксклюзивный тур премиум-класса по программе VIP+ с личным гидом!»
Маркетинговый сленг, «эксклюзивный/премиум/VIP» — пустые слова
«Прикоснитесь к древней мудрости Востока в стране махараджей и слонов 🐘🕌»
Стереотипы + emoji — отталкивает 30–50 ЦА

Рекомендация для [12.D2]–[12.D6]

  • Hero ([12.D2]): композит Black Tomato + Original Travel — full-bleed фото с тёмным gradient-overlay снизу, eyebrow «Южная Индия · 12 дней», headline в одну-две строки в стиле «отрицанием», CTA — текстовая ссылка с saffron-подчёркиванием, без plate-кнопки.
  • Day-by-day ([12.D3]): вертикальный timeline по образцу Original Travel. Каждый день — нумерованный кружок saffron, фото 40%, описание 60%, под описанием «Facts» как chips (расстояние / часы в дороге / высота).
  • Reviews ([12.D4]): текстовые карточки в стиле Much Better Adventures — фото-аватар + имя + город + 2–3 строки. Никаких видео-кружков на лендинге (это для post-trip Дневника).
  • Price + Lead ([12.D5]): price-block в стиле Intrepid («от X / на чел.»), но без агрессии. LeadForm — 3 поля (имя, email, telegram/whatsapp) + consent-чекбокс с ссылкой на политику данных, без галочки по умолчанию.
  • Mobile ([12.D6]): Much Better Adventures как референс — 100% mobile-first, sticky CTA внизу экрана с текстом «Узнать стоимость», hero сжимается до 70vh.
  • Photo brief ([12.D7]): Шиваму — снимать тёмные/контрастные кадры (закаты, силуэты лодок, чайные плантации в облаках), избегать ярко-полуденных Bollywood-кадров. Aspect 3:2 + 1:1 для мобильных.
