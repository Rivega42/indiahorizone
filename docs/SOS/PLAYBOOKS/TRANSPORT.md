# SOS Playbook — транспортный ЧП

> Закрывает [#25](https://github.com/Rivega42/indiahorizone/issues/25). Часть EPIC 3 [#19](https://github.com/Rivega42/indiahorizone/issues/19).
> Статус: Draft v0.1.

## Скоуп

Покрывает: ДТП с трансфером, поломка нашего трансфера, отмена/задержка авиарейса, отмена поезда, потеря багажа.

Граница: авиабилеты клиент покупает сам (см. [`docs/BUSINESS_MODEL.md`](../../BUSINESS_MODEL.md)). При отмене рейса — мы помогаем советом и пересборкой маршрута, но не возмещаем стоимость билета.

## Скрипт первого ответа

> «Я с вами. Имя — <X>. Скажите:
> 1. Что произошло — авария / поломка / отмена?
> 2. Все целы?
> 3. Где вы сейчас и куда нужно попасть?»

## Шаги реакции

### ДТП с нашим трансфером

1. **Все целы?** Если травмы — переход к [`MEDICAL.md`](./MEDICAL.md), параллельная реакция
2. Если все целы → фиксация: фото, контакты второй стороны, протокол
3. Резервный трансфер от партнёра (см. базу ниже) — отправляем
4. Связываемся со страховой нашей IN PVT LTD
5. Гид клиента (если в смене) выезжает координировать

### Поломка трансфера без ДТП

1. Резервный трансфер ASAP — приоритет в зависимости от того, опаздывает ли клиент на следующий рейс/мероприятие
2. Если опаздывают на критичный рейс → план Б (другой трансфер, перенос рейса, гостиница)
3. Запись простоя — для расчёта штрафа поставщику

### Отмена / большая задержка авиарейса

1. Информируем клиента: «есть подтверждение?»
2. Связываемся с авиакомпанией клиента (но клиент сам — мы помогаем советом)
3. Пересобираем наземный маршрут: пересдвиг гостиниц, экскурсий, трансферов
4. Если нужно — гостиница в аэропортовом городе на ночь
5. Уведомляем гида о новом времени прилёта

### Отмена поезда

1. Возврат билета — клиент сам через IRCTC, мы координируем
2. Альтернатива: автобус, машина, перенос рейса
3. Пересобираем маршрут

### Потеря багажа

1. Регистрация претензии в авиакомпании (PIR — Property Irregularity Report)
2. Помощь с переводом и заполнением
3. Минимальная аптечка / зубная щётка / переходник из аптеки рядом
4. Связь со страховой клиента (страховка часто покрывает)

## Сетка компенсаций клиенту (предложение)

| Ситуация | Что делаем |
|---|---|
| ДТП трансфера, всё ок | Извинение + сертификат на следующую поездку |
| Опоздание на критичный рейс по нашей вине | Перевыставление трансферов + покрытие 100% разницы билета |
| Поломка нашего трансфера, без последствий | Извинение в чате |
| Отмена рейса (не наша вина) | Помощь без компенсации, перестройка наземки бесплатно |
| Потеря багажа (не наша вина) | Координация с авиакомпанией бесплатно |
