## Резерв при отпуске / болезни

1. Расписание отпусков планируется на квартал вперёд
2. Минимум 7 дней предупреждения
3. Перекрытие через resourse pool (другой concierge берёт смену с доплатой)
4. При экстренной болезни — founders берут смену лично
