# Чек-лист обязательных документов клиента к вылету

> Закрывает [#42](https://github.com/Rivega42/indiahorizone/issues/42). Часть EPIC 4 [#31](https://github.com/Rivega42/indiahorizone/issues/31).
> Статус: Draft v0.1.

## Принцип

Без полного пакета документов **в Trip Dashboard и оффлайн-копии** — поездка не стартует. Менеджер несёт ответственность за полноту чек-листа.

## Чек-лист

### Обязательно для всех клиентов

| Документ | Срок проверки | Источник | Проверить |
|---|---|---|---|
| Паспорт РФ (главная страница) | ≥ 21 день до вылета | Скан от клиента | Срок действия > 6 мес после возвращения |
| Паспорт РФ (страница с визой IN) | ≥ 14 дней | Скан от клиента | Виза в порядке |
| Авиабилет / e-ticket | ≥ 14 дней | PNR от клиента | Имя совпадает с паспортом |
| Полис медицинской страховки | ≥ 14 дней | PDF от клиента | Покрывает Индию + сумма ≥ $30 000 |
| Ваучеры на отели | ≥ 7 дней | Из системы IH | Имя клиента, даты |
| Программа дня | ≥ 7 дней | Из Trip Dashboard | Сверена с гидом |
| Контакты экстренные | до подписания договора | Квиз | ФИО + телефон родственника |
| Согласия (ПДн, фото, гео, экст.контакты) | до подписания договора | Trip Dashboard | Все 4 даны |

### Дополнительно по обстоятельствам

| Случай | Документ | Кто отвечает |
|---|---|---|
| Ребёнок до 18 без обоих родителей | Согласие на выезд + апостиль | Клиент + sales |
| Ребёнок без родителей вообще | Доверенность | Клиент + юрист |
| Инвалидность / маломобильность | Медсправка для авиакомпании | Клиент |
| Особые лекарства | Рецепт + перевод | Клиент |
| Йога-ретрит | Документы от организатора | Клиент |
| Поход / трекинг | Расписка о понимании рисков | Клиент + IH |
| Беременность | Справка от врача о допуске к перелёту | Клиент |

## Регламент проверки

**Owner:** sales, ведущий клиента.

**За 21 день до вылета** — финальная сверка чек-листа. Если что-то отсутствует:

1. Срочный контакт с клиентом
2. Если клиент не может предоставить за 14 дней — эскалация Roman
3. Если за 7 дней документ всё ещё отсутствует — **поездка под угрозой**, обсуждение переноса

## Что делать если документа нет в день вылета

| Документ | Действие |
|---|---|
| Скан паспорта (но оригинал у клиента) | Делаем сами в аэропорту, добавляем в Trip Dashboard |
| Полис страховки | Помогаем оформить ускоренный полис онлайн (10 минут) |
| Виза | Поездка отменяется, форс-мажор, частичный возврат |
| Программа / ваучер | Распечатываем срочно или показываем с экрана |

## Хранение документов

- В Trip Dashboard клиента — **зашифровано** (поля `passport_*`, `visa_*` шифруются на стороне `clients-svc`)
- Доступ к скану паспорта — только клиент, sales, concierge на смене (audit-log записывает каждый просмотр)
- Срок хранения: см. [`docs/LEGAL/PDN.md` § Retention](../LEGAL/PDN.md)
- Скан паспорта удаляется через 6 месяцев после возвращения
