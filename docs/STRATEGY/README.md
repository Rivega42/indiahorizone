## Контекст

**Сейчас (фаза 3):**
- Платящие клиенты есть
- ТЗ написано на основе реальных болей
- Процессы на 90% — WhatsApp + Notion + Google Docs
- Платежи: РФ юрлицо → IN юрлицо вручную
- SOS держится на одном человеке (Шивам)

**Что это значит:**
Бизнес работает. Фундамент заложен. Следующий шаг — не «пилить фичи», а выбрать вектор масштабирования и начать его строить параллельно с операционкой.

## Выбранный вектор

**Вариант 1 + Franchise/White-label OS** (см. [`VISION.md`](./VISION.md)).

Кратко: IndiaHorizone → методология + платформа «ведения» клиента в нишевом travel.
Не маркетплейс. Не глобальный агрегатор. **Операционная система для нишевых туроператоров.**
