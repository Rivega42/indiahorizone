# 🏗️ Архитектура — IndiaHorizone

## Компоненты системы

```
┌─────────────────────────────────────────────────────┐
│                 Клиентские интерфейсы               │
│  [Мобильное приложение]  [Веб-портал]  [Admin]      │
└──────────────────────────┬──────────────────────────┘
                           │ HTTPS / WebSocket
┌──────────────────────────▼──────────────────────────┐
│                    API Gateway                      │
│           NestJS + авторизация (JWT/RBAC)           │
└────┬──────────┬──────────┬──────────────────────────┘
     │          │          │
  Trips      CRM      Feedback
  Service   Service   Service
     │          │          │
┌────▼──────────▼──────────▼────┐
│        PostgreSQL             │
│   Redis (кеш + очереди)       │
│   S3 (файлы, фото, видео)     │
└───────────────────────────────┘
```

## Роли

| Роль | Описание | Интерфейс |
|------|----------|-----------|
| `client` | Путешественник | Мобильное приложение + PWA |
| `guide` | Гид / локальный оператор | Мобильное приложение |
| `concierge` | Поддержка 24/7 | Веб + мобильный |
| `manager` | Менеджер по продажам | CRM (веб) |
| `finance` | Финансы | Веб |
| `admin` | Управляющий | Полный доступ |

## Технологический стек

| Слой | Стек |
|------|------|
| Frontend | Next.js 14 + React 18 + TypeScript |
| Mobile | React Native + Expo |
| UI | Tailwind CSS + shadcn/ui |
| Backend | NestJS + Prisma |
| БД | PostgreSQL 15+ |
| Кеш | Redis 7+ |
| Файлы | S3-совместимое хранилище |
| AI | OpenAI GPT-4o |
| Push | FCM (Android) + APNs (iOS) |
| Инфра | Docker Compose + GitHub Actions |

## Структура репозитория

```
indiahorizone/
├── apps/
│   ├── web/          # Next.js клиентский портал
│   ├── mobile/       # React Native
│   ├── admin/        # Панель управляющих
│   └── api/          # NestJS backend
├── packages/
│   ├── ui/           # Общие UI компоненты
│   ├── core/         # Бизнес-логика
│   ├── notifications/ # Уведомления
│   └── ai/           # AI-интеграции
├── docs/             # ТЗ, архитектура, роадмап
├── .github/
│   └── workflows/    # CI/CD + автоматизация
└── docker-compose.yml
```

## Ключевые принципы

- **Offline-first** — критичные данные кешируются локально
- **Mobile-first** — основные сценарии для смартфона
- **Русскоязычный UX** — весь интерфейс на русском
- **Минимум шагов** — SOS, фидбэк — максимум 2 тапа
