# 📖 Глоссарий — IndiaHorizone

| Термин | Определение |
|--------|-------------|
| **Trip Dashboard** | Личный кабинет клиента на время поездки |
| **Concierge** | Русскоязычный специалист поддержки 24/7 |
| **SOS** | Экстренная кнопка мгновенной связи с concierge |
| **Daily Feedback** | Ежедневный отзыв клиента (текст / видео-кружок) |
| **Guide** | Локальный гид / оператор в Индии |
| **Milestone (M)** | Крупный этап разработки (M1–M5) |
| **Epic** | Группа связанных задач (EPIC 7, 8, 9, 10) |
| **P0–P3** | Приоритет задачи (P0 = критический) |
| **PWA** | Progressive Web App — веб-приложение с оффлайн-режимом |
| **RBAC** | Role-Based Access Control — управление доступом по ролям |
| **LTV** | Lifetime Value — пожизненная ценность клиента |
| **CAC** | Customer Acquisition Cost — стоимость привлечения клиента |
| **NPS** | Net Promoter Score — индекс лояльности клиентов |
| **INR** | Индийская рупия |
| **OCPlatform** | AI-платформа на которой работает ассистент команды |
