## Продукт

### Что мы продаём

**Кастомный маршрут по Индии под клиента (или малую группу 2–6 человек) с наземным сопровождением.**

В стоимость входит:

- Консультация и подбор маршрута (1–N итераций до утверждения)
- Бронирование отелей в Индии
- Бронирование трансферов (аэропорт, между городами, локально)
- Бронирование экскурсий и активностей
- Услуги гидов (русскоязычных и местных)
- Сопровождение от Trip Dashboard
- Concierge 24/7 на всём протяжении поездки
- SOS-линия с SLA (см. [`docs/SOS/CONCEPT.md`](./SOS/CONCEPT.md))
- Финальный «дневник поездки» (фото + видео-кружки)

### Формат «ведения» клиента

| Этап | Касание | Канал |
|---|---|---|
| Лид → квалификация | Звонок / переписка | Telegram, WhatsApp |
| Квиз | Самостоятельно клиентом | Веб |
| Подбор маршрута | Совместная работа menager↔клиент | Telegram + Notion / Google Doc |
| КП | Готовый документ + созвон | Email + Zoom |
| Договор + предоплата | Договор-оферта | Email + платёжный шлюз |
| Активная подготовка | Чек-листы клиенту, советы | Trip Dashboard + чат |
| За N дней до старта | Финальный созвон | Zoom / Telegram |
| В поездке | Concierge + гид | Trip Dashboard + WhatsApp |
| Пост-поездка | Фидбэк, дневник, реферал | Trip Dashboard + email |

### Группы клиентов

**Индивидуально (1–2 человека):** наш основной формат фазы 3.

**Малая группа (2–6 человек):** друзья / семья. Цена — индивидуальная, не «на человека».

**Большие группы (7+):** не делаем в фазе 3 (требует другой операционки).

**Тематические туры:** только если запрошены клиентом (например, «йога в Ришикеше»). Не отдельный продукт.

### Что мы НЕ продаём (в фазе 3)

- Авиабилеты — рекомендуем, но не продаём
- Страховку — рекомендуем, но не продаём
- Визу в Индию — помогаем советом

## Граница ответственности

### По авиабилетам

| Ситуация | Что делаем |
|---|---|
| Клиент просит совет по выбору рейса | Рекомендуем направление и поставщика |
| Изменение / отмена рейса до поездки | Помогаем пересобрать наземку, ничего не возмещаем |
| Опоздание клиента на пересадку | Помогаем найти выход (отель в транзите, новый рейс), ничего не возмещаем |
| Овербукинг / поломка самолёта | Координируем с авиакомпанией клиента, помогаем составить претензию, не возмещаем |

### По страховке

| Ситуация | Что делаем |
|---|---|
| Клиент просит совет по выбору страховки | Рекомендуем страховую компанию и план |
| Страховой случай в Индии | Координируем с ассистансом страховой клиента, не платим за лечение |
| Страховая отказала | Помогаем составить претензию, не компенсируем разницу |

### По SOS

См. полный список «не покрывается» в [`docs/SOS/CONCEPT.md`](./SOS/CONCEPT.md).

Кратко:
- Не возмещаем стоимость билета / страховки
- Не оплачиваем лечение (страховая клиента)
- Не представляем в суде (находим юриста)
- Не работаем за пределами Индии

### По гидам и поставщикам

Мы отвечаем за **подбор и контроль качества** наших партнёров (отели, гиды). Если партнёр подвёл — мы:
- Переселяем / меняем
- Компенсируем по сетке (см. [`docs/SOS/PLAYBOOKS/TRANSPORT.md`](./SOS/PLAYBOOKS/TRANSPORT.md))
- Получаем штраф у партнёра (наше внутреннее)

Мы НЕ отвечаем за форс-мажор партнёра (например, забастовка такси), но включаемся для пересборки.
