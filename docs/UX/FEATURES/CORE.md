## Принцип

Ядро = то, что доступно **оффлайн** и работает за 2 тапа.

| Раздел | Цель |
|---|---|
| Документы и маршрут | Клиент в Индии без сети открывает паспорт / билет / ваучер |
| Таймлайн дня | «Сейчас» и «Следующее» — не нужно искать |
| Фидбэк (текст + кружок) | Один тап от пуша до записи |
| SOS | Hold-to-trigger, всегда виден |

## Раздел: Документы и маршрут

### Состав

- **Паспорт** (скан с визой) — открывается в полноэкранном просмотре
- **Авиабилеты** — туда / обратно
- **Полис страховки** — PDF + краткие контакты ассистанса
- **Ваучеры на отели** — по дням
- **Полная программа** — день за днём, оффлайн карта

### UX

- Все файлы — открываются **без интернета** (предзагружены)
- Карты с маршрутами — оффлайн (Mapbox с кешем региона)
- Поделиться отдельным документом → нативный share sheet
- Если документ обновился — push «обновили программу на 5 день», клиент видит badge

См. [`docs/ARCH/OFFLINE.md`](../../ARCH/OFFLINE.md), [#68](https://github.com/Rivega42/indiahorizone/issues/68):

## Раздел: Таймлайн дня

### Структура

- **Сейчас** — текущая активность (по локальному времени)
- **Следующее** — ближайшее с таймером до начала
- **Полный день** — расписание с визуальным прогрессом

### Поведение

- Push-напоминание за 2 часа до выезда / трансфера
- Push «через 30 минут» о ключевом событии
- Каждое событие — кликабельно: детали (адрес, телефон, заметки гида)

## Раздел: Фидбэк (текст + кружок)

### Сценарий

19:00 локально → push: «Как прошёл день <X>?»

Тап → экран с двумя кнопками:

```
[  📝 Написать текстом  ]
[  🎬 Записать кружок  ]
```

Текстом: textarea + 5 emoji-оценок (😔 😐 🙂 😊 🤩) → отправить.

Кружком: см. [`docs/UX/VIDEO_CIRCLE.md`](../VIDEO_CIRCLE.md), [#85](https://github.com/Rivega42/indiahorizone/issues/85).

### После отправки

- В UI на главном экране бывший кружок виден как «✓ День 3 — отправлено»
- Concierge увидит → может ответить в чат, если что-то нехорошо

## Раздел: SOS

### Размещение

- Всегда виден на главном экране
- На других экранах — floating button в нижнем правом углу
- На системном уровне — quick action (long-press на иконке приложения для iOS / shortcuts для Android)

### Поведение

- **Hold 2 секунды** на кнопке (анти-FP)
- 10-секундное окно для отмены
- См. [`docs/SOS/FALSE_POSITIVE.md`](../../SOS/FALSE_POSITIVE.md), [#29](https://github.com/Rivega42/indiahorizone/issues/29)

### При активации

- Заметный экран с «Помощь идёт. Имя дежурного — Х. Время на ответ ~60 сек»
- Открыт чат с concierge
- Геолокация автоматически передана
